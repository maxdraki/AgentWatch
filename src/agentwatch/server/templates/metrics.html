📖 Recording Metrics

Track custom numeric values from your agent code:

import agentwatch

agentwatch.init("my-agent")

# Record a gauge (point-in-time value)
agentwatch.metric("queue_depth", 42)

# Record with tags for filtering
agentwatch.metric("requests", 1, tags={"method": "POST"})

# Record a counter
agentwatch.metric("errors_total", 5, kind="counter")

# Query metrics
from agentwatch.metrics import query, summary
points = query("queue_depth", hours=24)
stats = summary("queue_depth", hours=1)

Gauges represent point-in-time values (e.g. queue depth, memory usage).

Counters represent monotonically increasing values (e.g. request count, total errors).
